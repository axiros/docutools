1""" 

2## Markdown Replace 

3 

4Allows to add replacements into the markdown source, which are replaced by values given 

5by a python module. 

6 

7The python module (default is: docs/mdreplace.py) must have a table attribute, either 

8dict or callable. 

9 

10When callable and kw args in the signature, it will be called with a lot of context, incl. the mkdocs config and the 

11current line. Otherwise it will be simply called. 

12 

13 

14### Features 

15 

16- The replace values can themselves be callable, and if so, are called at replacement 

17 time with contextual information:  

18 

19 ```python 

20 replace( 

21 mdblock=md, 

22 plugin=self, 

23 plugin_file=__file__, 

24 config=config, 

25 page=page, 

26 markdown=markdown, 

27 ) 

28 ``` 

29 

30- If the callable does not require kw args (e.g. time.ctime) we will not pass them 

31- The callable can return a replacement for the whole line, by returning a dict like 

32 `{'line': ....}`, i.e. with a "line" key. 

33- If the replace values are lists (also as returned by the callable), they will be 

34 properly indented as multiline text. 

35 

36#### Controlling Replacements Within Fenced Blocks 

37 - fenced blocks are omitted EXCEPT: 

38 - if the replacement key is specified like this `key:all:` - then even `:key:` in 

39 fenced blocks is replaced 

40 

41### Config 

42 

43- `seperator`: ':' by default.  

44 Example: `':curtime:'`, for `{"cur_time": time.ctime}` based replacements. 

45- `replacement_file`: when not starting with '/' we'll prefix with docs_dir. Default: "mdreplace.py" 

46"""
